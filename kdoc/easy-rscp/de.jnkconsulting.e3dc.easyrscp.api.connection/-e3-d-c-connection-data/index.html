E3DCConnectionData

data class E3DCConnectionData(val address: String, val port: Int = 5033, val portalUser: String, val portalPassword: String, val rscpPassword: String, val timeoutMillis: Long = 10000)

Simple data class to provide the connection data to the home power plant

Since

2.0

Parameters

address

IP address or DNS name of the home power plant

port

Port on which the power plant listens. Default: 5033 (Info: 5034 for Farming)

portalUser

Username. Corresponds to the username from the E3DC portal

portalPassword

Password. Corresponds to the password on the E3DC portal

rscpPassword

Encryption password. This value is configured directly at the home power plant and must be identical here.

timeoutMillis

Milliseconds for the timeout. Default 10000
