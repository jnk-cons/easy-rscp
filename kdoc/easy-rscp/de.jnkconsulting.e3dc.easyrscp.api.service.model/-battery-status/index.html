BatteryStatus

data class BatteryStatus(val index: Short, val trainingModeActive: Boolean, val connected: Boolean, val working: Boolean, val inService: Boolean, val asoc: Float, val realRsoc: Float, val voltage: Float, val dcbStatus: List<DCBStatus>)

Contains information on the current status of the battery

Since

2.1

Parameters

index

ID of the battery in the E3DC system

trainingModeActive

Information on whether the battery is in training mode

connected

Information on whether the battery is connected to the system (battery disconnect switch on the home power station)

working

Information on whether the battery is working correctly

inService

Information on whether the battery is in maintenance mode

asoc

Percentage value that includes the condition of the battery taking ageing into account. A new battery theoretically has a value of 1.0 (100%)

realRsoc

Current charge level as a percentage of the battery without taking into account the reserve for absolute deep discharge and theoretically possible full charge. The value should therefore never reach completely 0 and never completely 1

voltage

Current battery voltage

dcbStatus

Status of the individual battery modules
